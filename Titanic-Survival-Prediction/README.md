# Titanic Survival Prediction

Machine learning model to predict passenger survival using the Titanic dataset.
Includes preprocessing, EDA, model training, evaluation, and result generation.

## Files
- titanic_survival_prediction.py — main script
- train.csv — training dataset
- test.csv — test dataset
- resultfile.csv — prediction output
- gender_submission.csv — sample submission file

## Tech Stack
Python, Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
